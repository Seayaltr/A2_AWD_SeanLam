# Assignment 2 - Airline Booking System

## GitHub Repository

https://github.com/Seayaltr/A2_AWD_SeanLam

## Local Setup

1. Run `npm install`
2. Add MongoDB details in `.env.local`
3. Run `npm run seed`
4. Run `npm run dev`
5. Open `http://localhost:3000`
